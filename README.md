# judge_json
An Elixir rule engine where rules are json objects. The judge gives verdicts on any rule matches for a provided payload.


## Installation

If [available in Hex](https://hex.pm/docs/publish), the package can be installed

```elixir
def deps do
  [
    {:judge_json, "~> 0.1.0"}
```
